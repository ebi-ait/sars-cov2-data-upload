Obtain a secure submission key

To use the data uploader you need to request a secure submission key, by contacting our helpdesk - virus-dataflow@ebi.ac.uk. In your request, please provide an email address, and, if already created, a Webin submission account. Please let us know if you would like a new Webin account created. A user support bioinformatician will contact you and provide you with a secure submission key within a working day.

Can I share my submission key? The secure submission key uniquely identifies you and serves as authentication to the uploader file - do not share your submission key unless you want to share file access.

Login using a secure submission key

When reaching the data uploader web page, you will use your secure submission key to login into the system. Enter your key in the input field, and click the login/ validate button. I entered the wrong key? If your submission key is invalid an error message will be displayed.

My submission key is not working? Please email our helpdesk virus-dataflow@ebi.ac.uk who will be able to assist.
